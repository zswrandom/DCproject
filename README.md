# 校汇通 (Campus-Connect) - 智能视频会议系统

本项目是一个集实时视频通讯、AI 智能会议纪要及即时通讯于一体的视频会议简单模型。

## 🚀 项目功能展示（部分尚未完成）

- **实时视频预览**：调用本地媒体设备，实现高清视频流采集与展示，可点击摄像头按钮进行使用
- **AI 智能纪要分栏**：独立侧边栏设计，支持调用后端 AI 接口实时生成会议摘要。
- **即时聊天界面**：集成参与者列表与会议聊天窗口，可以实现即时聊天

## 📂 文件说明

-`打开front文件中的三个文件，下面是对三个文件的介绍
- `index.html`: 项目核心结构文件，实现页面整体布局。
- `camera.js`: 负责媒体设备（摄像头、麦克风）的调用、流控制及状态切换。
- `api.js`: 负责与后端 AI 分析接口通讯，处理请求状态及数据渲染。

## 🛠 如何运行

1. **环境准备**：
   - 确保上述三个文件位于同一级目录下。
   - 由于浏览器安全策略，调用摄像头功能需在 **HTTPS** 环境或 **Localhost** 下运行。

2. **启动项目**：
   - 推荐使用 VS Code 的 `Live Server` 插件点击右下角 "Go Live" 或者 安装插件后点击鼠标右键，点击出现的Open with Liver Server运行。
   - 也可以在pycharm中选择合适的浏览器图表运行

3. **后端连接**：
   - 本项目默认对接 `http://127.0.0.1:8001` 的 AI 接口。
   - 运行前请确保后端服务已启动：`uvicorn main:app --reload --port 8001`。
